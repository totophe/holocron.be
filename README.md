# Holocron - Journey Through the Stars

A Star Wars-themed interactive webpage featuring a beautiful 3D star tunnel effect with hyperspace travel capability.

## Features

- **3D Star Tunnel Effect**: Stars radiate from the center and travel outward, creating an immersive tunnel experience
- **Safe Zone**: Stars maintain a minimum distance from the center, preventing the feeling of traveling through them
- **Hyperspace Mode**: Toggle between cruising speed and hyperspace with dramatic visual effects
- **Responsive Design**: Adapts to different screen sizes
- **Keyboard Control**: Press SPACE to toggle hyperspace mode

## Technologies

- HTML5 Canvas for rendering
- Pure JavaScript (no frameworks)
- CSS3 animations and effects
- Custom 3D perspective projection

## How to Use

1. Open `index.html` in a modern web browser
2. Watch the stars slowly drift past in normal cruising mode
3. Click the "HYPERSPACE" button (bottom right) or press SPACE to engage hyperspace
4. Press again to disengage and return to normal speed
